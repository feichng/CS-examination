# 题目

打印九九乘法表

1 * 1 = 1

2 * 1 = 2 2 * 2 = 4

......

9 * 1 = 9 ...... 9 * 9 = 81

# 答案

```java
class NineNineTable {
    public static void main(String[] args) {
        
        for(int i = 1;i <= 9;i++){
            
            for (int j = 1;j <= i;j++){
                System.out.println(i + " * " + j + " = " + (i * j) + " ");
            }
            
            System.out.println();
        }
    }
}
```
